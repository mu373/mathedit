# mathedit
Convert LaTeX math equations to SVG with embedded metadata for round-trip editing.

https://mathedit.vercel.app/

## Features
- Round-trip editing: The SVG files generated include metadata including original LaTeX equations, which allows user to re-import and re-edit.
- Color: Supports `\color{}` tags
- Project saving: Save sets of equations as a project
